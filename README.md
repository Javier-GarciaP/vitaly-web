# Vitaly Lab Management

Sistema integral de gestión para laboratorios clínicos, diseñado para optimizar el flujo de trabajo, el registro de pacientes y la entrega de resultados.

![Dashboard Preview](docs/images/dashboard.png)

## Características Principales

- **Gestión de Pacientes**: Registro detallado y seguimiento de historial clínico.
- **Control de Exámenes**: Procesamiento de hematología, química sanguínea, coagulación y más.
- **Reportes Profesionales**: Generación de resultados en PDF con diseño minimalista y códigos QR de verificación.
- **Panel de Control**: Visualización de estadísticas y tendencias en tiempo real.
- **Multi-plataforma**: Disponible como aplicación web y de escritorio (Tauri).

## Tecnologías Utilizadas

- **Frontend**: React 19, Vite, Tailwind CSS.
- **Backend/Base de Datos**: Cloudflare Workers, D1 (SQLite), Firebase Auth.
- **Desktop**: Tauri (Rust/React).
- **Reportes**: React-PDF.
